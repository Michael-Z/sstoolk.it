SSBadgeView Class Reference

Declared in SSBadgeView.h

Overview

Badge view.

Provides the accesory view for a SSBadgeTableViewCell, available through the badgeView property.

Acts very much like the badges in Mail.app, with the key difference being that Apple uses images and SSBadgeView is rendered with CoreGraphics for improved scrolling performance.

Not limited to numbers, strings should still be short enough to fit within the accesoryView bounds.
