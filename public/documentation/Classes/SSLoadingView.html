SSLoadingView Class Reference

Declared in SSLoadingView.h

Overview

easy view to show loading similar to the loading screens in the iTunes or App Store applications. The view is intended to cover another view while it is loading and the removed once loading is complete.

This view will automatically center the text and the activity indicator.

shadowOffset

The shadow offset (measured in points) for the text.

@property CGSize shadowOffset
Discussion

The shadowColor must be non-nil for this property to have any effect. The default offset size is (0, 1), which indicates a shadow one point below the text. Text shadows are drawn with the specified offset and color and no blurring.
