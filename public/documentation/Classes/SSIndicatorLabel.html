SSIndicatorLabel Class Reference

Declared in SSIndicatorLabel.h

Overview

This class is a nice little view that makes showing an activity indicator next to a label when loading something easy.

The indicator will slide in from the left when loading and then back out when it is complete. This is great for a view that shows some sort of status that can refresh.
