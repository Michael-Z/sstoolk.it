dateFromISO8601String:

Returns a new date represented by an ISO8601 string.

+ (NSDate *)dateFromISO8601String:(NSString *)iso8601String

Parameters

iso8601String

An ISO8601 string

Return Value

Date represented by the ISO8601 string

Discussion

Issue #56: Currently this method doesn’t work with strings that include a timezone as anything other than “Z” like this one “2011-02-01T10:57:55-08:00”.

timeAgoInWordsFromTimeInterval:includingSeconds:

Returns a string representing a time interval in words.

+ (NSString *)timeAgoInWordsFromTimeInterval:(NSTimeInterval)intervalInSeconds includingSeconds:(BOOL)includeSeconds

Parameters

intervalInSeconds

The time interval to convert to a string

includeSeconds

YES if seconds should be included. NO if they should not.

Return Value

A string representing the time interval in words

Discussion

The strings produced by this method will be similar to produced by ActiveSupport’s time_ago_in_words helper method.
