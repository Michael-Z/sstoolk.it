Properties

alpha

The receiver’s alpha value. (read-only)

@property (nonatomic, assign, readonly) CGFloat alpha

Discussion

The value of this property is a floating-point number in the range 0.0 to 1.0, where 0.0 represents totally transparent and 1.0 represents totally opaque.

blue

The receiver’s blue component value. (read-only)

@property (nonatomic, assign, readonly) CGFloat blue

Discussion

The value of this property is a floating-point number in the range 0.0 to 1.0. -1.0 is returned if the color is not in the RGB colorspace.

hexValue

Returns the receiver’s value as a hex string.

- (NSString *)hexValue

Return Value

The receiver’s value as a hex string.

The value will be nil if the color is in a color space other than Grayscale or RGB. The # sign is omitted.
